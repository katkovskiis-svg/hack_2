# Предсказание конверсии на сайте «СберАвтоподписка»

## Описание проекта

Проект решает задачу предсказания вероятности совершения пользователем целевого действия (оставить заявку, заказать звонок, открыть диалог) на сайте сервиса долгосрочной аренды автомобилей «СберАвтоподписка».

**Бизнес-ценность:**
- Оценка эффективности каналов привлечения трафика
- Адаптация и оптимизация рекламных кампаний
- Улучшение пользовательского опыта (UX) на основе анализа поведения

**Результат:** Модель CatBoost с ROC-AUC = **0.9767** (при ориентире 0.65).

## Данные

Данные из Google Analytics (last-click attribution model) по сайту «СберАвтоподписка»:

| Файл | Описание | Строк | Столбцов |
|------|----------|-------|----------|
| `ga_sessions.pkl` | Визиты на сайт (одна строка = один визит) | ~1 860 000 | 18 |
| `ga_hits.pkl` | События в рамках визитов (одна строка = одно событие) | ~15 726 000 | 11 |

**Ключевые атрибуты:**
- **Сессии:** `session_id`, `client_id`, UTM-метки (`utm_source`, `utm_medium`, `utm_campaign`), устройство (`device_category`, `device_os`), география (`geo_country`, `geo_city`)
- **Хиты:** `session_id`, `event_action` (тип действия пользователя), `hit_page_path` (страница)

## Установка

- pip

```bash
pip install -r requirements.txt
```

## Запуск

### 1. Jupyter Notebook (аналитический отчет)

```bash
jupyter notebook notebooks/analysis.ipynb
```

Notebook содержит:
- Загрузку и очистку данных
- Разведочный анализ (EDA) с графиками
- Обучение и оценку ML-моделей
- Выводы и рекомендации

### 2. Обучение модели (отдельно)

```bash
python app/model.py
```

Скрипт загружает данные, обучает CatBoost и сохраняет модель в `models/model.pkl`.

### 3. Запуск API

```bash
uvicorn app.app:app --host 0.0.0.0 --port 8000
```

API будет доступен по адресу: http://localhost:8000

**Документация API (Swagger):** http://localhost:8000/docs

### Пример запроса к API

```bash
curl -X POST http://localhost:8000/predict \
```

## Результаты

### Метрики моделей

| Модель | ROC-AUC |
|--------|---------|
| Logistic Regression (baseline) | 0.9308 |
| **CatBoost** | **0.9767** |

### Ключевые выводы из EDA

1. Общая конверсия сайта ~2% — типичное значение для автомобильной тематики
2. Desktop-пользователи конвертируются лучше мобильных
3. Поведенческие признаки (просмотр карточек авто, quiz) сильно связаны с конверсией
4. Различные каналы привлечения показывают разную эффективность

### Топ-5 важных признаков модели

Наиболее значимые факторы для предсказания конверсии — поведенческие характеристики сессии (количество событий, просмотр карточек) и характеристики канала привлечения.
